# ood-device-sdk

#### 介绍
常见监控设备SDK接口工具，目前主要对接海康、大华，后续对接更多。

#### 软件架构
引用lombok、hutool、jna、第三方库

#### 使用说明

1.  海康SDK，采用6.1.6.45版本，支持win64和linux。通过HikDeviceUtil获取海康设备SDK
